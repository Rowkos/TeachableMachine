About Yahtzee Detector:

   A Pointless Project by Rowan Koskoff


This is intended as a demonstration of the video recognition software Teachable Machines. My main goal is to demonstrate that Image recognition software can be used to detect very similar ideas like different faces of a dice. Yahtzee was added to provide a fun twist. There are myriad of ways it could be improved, for example: using a separate webcam to actually look at dice being rolled; rolling multiple dice and using image recognition to split that image into multiple sub-images each containing a single dice; and allowing the user to play a full game of Yahtzee, not just tell them what their best move is.
